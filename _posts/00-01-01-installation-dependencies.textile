h2. Python modules

* numpy >= 1.6.0
* scipy
* matplotlib >= 0.99
* "pyfftw":https://launchpad.net/pyfftw/ >= 0.2.1 (compiled with options @--enable-threads@ and @--enable-shared@)
* pyfits >= 3.0
* kapteyn >= 2.0.2

<h3>Optional:</h3>
* mpi4py


h2. Other libraries

* gfortran >= 4.5 or ifort >= 11.1
* cfitsio >= 3.23
* FFTW >= 3.2.2

<h3>Optional:</h3>

* wcslib >= 4.4.4
* openmpi or mpich
